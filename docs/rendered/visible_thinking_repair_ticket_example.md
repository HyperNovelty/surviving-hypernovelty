# Visible Thinking Repair Ticket: Research paragraph missing source trail

**Status:** REPAIR NEEDED  
**Owner:** Teacher / learning reviewer  
**Purpose:** Give a learner a low-conflict repair path when submitted work lacks enough visible thinking evidence.  
**Review cadence:** Per assignment or after a weak-evidence submission.

## Missing evidence
- Which source supports the main claim is unclear (severity: high)
- Student reasoning between quote and conclusion is not visible (severity: high)
- AI-use disclosure says "grammar only," but revision trail is absent (severity: medium)

## Repair request
- Explain the paragraph claim aloud or in 5-7 sentences in your own words (evidence type: oral_explanation)
- Mark which sentence is supported by which source note (evidence type: source_mapping)
- Submit a short revision note naming any AI help and what you accepted/rejected (evidence type: revision_note)

## Acceptable repair outcomes
- Evidence now supports the claim and learner can explain it -> accept
- Claim is narrowed or removed -> revise
- Teacher schedules a short conference if evidence remains unclear -> conference

## Review flags
- Not proof of authorship
- Do not use as detector evidence
- Use local educator judgment for grading/conduct decisions

## Boundary note
Review aid only. This is not forensic proof, misconduct determination, or detector validation. It is a learning-evidence repair workflow.
