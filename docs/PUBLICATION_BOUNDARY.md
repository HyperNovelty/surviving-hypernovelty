# Publication Boundary

This repository is an early public candidate for a local starter kit. It can be
prepared for public review, but it should not include private Hypernovelty
material or imply public adoption.

## Safe to Publish

- tool code that runs locally with the Python standard library
- schemas for the public-safe starter-kit examples
- synthetic examples under `examples/`
- repo-relative templates under `tools/`
- general documentation about how to use the tools
- rendered examples that are generated from synthetic fixtures
- validation workflow configuration for a future public repository

## Stays Private or Proprietary

- private Hypernovelty sources, transcripts, notes, or research vault material
- grant strategy, fundraising plans, customer pipeline, or pricing strategy
- customer data, student data, health data, legal/financial records, or support
  conversations
- credentials, tokens, account exports, browser sessions, or payment material
- internal filesystem paths outside this repository
- proprietary operating plans, institutional relationships, or unpublished
  strategy documents

## Public Claim Rules

Public-facing text should say this is an early public candidate and local starter
kit. It should not claim adoption, funding, certification, institutional approval,
customer validation, or production readiness.

Tool outputs are review aids. Any public, account, paid, customer-facing,
high-stakes, or proprietary use requires separate human approval.
