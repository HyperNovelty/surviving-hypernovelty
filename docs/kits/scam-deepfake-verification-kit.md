# Scam / Deepfake Verification Kit

**Public promise:** Slow down high-pressure impersonation, fraud, and bad source-chain decisions before people click, pay, share codes, or approve sensitive actions.

**Audience:** Families, school offices, nonprofits, small businesses, admins, finance/ops teams.

## Use this when

- A person or team has a concrete decision to make under fast-changing AI/information conditions.
- The output needs to be reviewable without uploading private data.
- A synthetic example is enough to teach the workflow before customization.

## Included primitives

- Scam Attention Receipt
- Verification Literacy Mini-Lab: Scam Edition
- Human Recourse Path Card v2 — scam-contact example
- Human Recourse Drill Card

## Candidate extension handling

Deepfake Approval Pause Card should start as a scam/deepfake example, not a new primitive.

## Boundary

Not fraud detection, cybersecurity certification, or financial/legal advice.

## Open-source release notes

- Keep examples synthetic.
- Keep browser-local/offline paths prominent.
- Do not imply legal, security, academic-integrity, medical, financial, or compliance authority.
- Paid value later must come from customization, facilitation, hosting, recurring review, or implementation support — not from guaranteed outcomes.
