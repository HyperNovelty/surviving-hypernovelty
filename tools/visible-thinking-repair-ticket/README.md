# Visible Thinking Repair Ticket

Repair missing learning evidence without relying on AI-detector false certainty or punitive defaults.

## Use

```bash
python3 render_ticket.py ../../examples/education/visible_thinking_repair_ticket_example.json
```

From the repo root:

```bash
python3 tools/visible-thinking-repair-ticket/render_ticket.py examples/education/visible_thinking_repair_ticket_example.json
```

## Boundary

Local-first review aid only. Do not use this as legal, compliance, security, medical, academic-integrity, or policy authority. Public posting, outreach, submissions, account changes, spend, real private data, and formal claims require separate human approval.
