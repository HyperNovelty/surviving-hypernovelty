# Workslop Review Receipt

A small local receipt for separating **generated/drafted** work from **reviewed/defensible** work.

Use it when an AI-assisted draft looks finished but downstream coworkers still need to know what was checked, what is weak, who owns it, and what should not be claimed.

Boundary: this is not an AI detector, HR discipline tool, quality guarantee, legal/compliance review, or worker-blame artifact.

```bash
python3 tools/workslop-review-receipt/render_receipt.py examples/team/workslop_review_receipt_example.json
```
