# Adaptation Debt Ledger

Name stale assumptions, symptoms, affected people, owners, cheap review actions, and top repair priorities.

## Use

```bash
python3 render_ledger.py ../../examples/institution/adaptation_debt_ledger_example.json
```

From the repo root:

```bash
python3 tools/adaptation-debt-ledger/render_ledger.py examples/institution/adaptation_debt_ledger_example.json
```

## Boundary

Local-first review aid only. Do not use this as legal, compliance, security, medical, academic-integrity, or policy authority. Public posting, outreach, submissions, account changes, spend, real private data, and formal claims require separate human approval.
