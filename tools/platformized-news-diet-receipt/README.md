# Platformized News Diet Receipt

Track where news came from this week: direct source, platform, creator, AI summary, confidence, and missing context.

## Use

```bash
python3 render_receipt.py ../../examples/media/platformized_news_diet_receipt_example.json
```

From the repo root:

```bash
python3 tools/platformized-news-diet-receipt/render_receipt.py examples/media/platformized_news_diet_receipt_example.json
```

## Boundary

Local-first review aid only. Do not use this as legal, compliance, security, medical, academic-integrity, or policy authority. Public posting, outreach, submissions, account changes, spend, real private data, and formal claims require separate human approval.
