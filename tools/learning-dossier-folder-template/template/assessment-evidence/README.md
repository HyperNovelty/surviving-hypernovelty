# Assessment Evidence Packet

A proof-of-learning packet for AI-era education. The goal is not to catch students using AI; the goal is to design better evidence of understanding.

## What belongs here

- The task or question being assessed.
- The learner's attempt.
- A short explanation in the learner's own words.
- A transfer problem or variant.
- Misconceptions found.
- Feedback received.
- Human/teacher/mentor review.

## Use cases

- Math/science problem solving.
- Writing/research assignments.
- Project-based learning.
- Tutoring or self-study.
- AI-assisted learning where the final answer alone is weak evidence.
