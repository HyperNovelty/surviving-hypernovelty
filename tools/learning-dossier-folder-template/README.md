# Learning Dossier Folder Template

“The Next Classroom May Be a Folder.”

This template makes learning state visible, durable, and reviewable. It is not just an AI tutor chat. It is a learner-owned workspace with evidence, misconceptions, assessment, sources, feedback, and next steps.

## Starter structure

Copy `template/` for each learner/project.

```text
template/
  mission.md
  learner-state.md
  sources.md
  questions.md
  attempts/
  feedback/
  assessments/
  misconceptions.md
  next-steps.md
  human-review.md
```

## Verification layer

Every learning dossier should eventually answer:

- What does the learner care about?
- What did they try?
- What evidence shows understanding?
- What misconception was found and corrected?
- What transfer problem can they solve?
- What did a human/teacher/mentor review?
