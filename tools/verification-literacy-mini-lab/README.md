# Verification Literacy Mini-Lab: Scam Edition

A 20–30 minute local lesson/workshop packet for practicing verification with synthetic scam-like messages.

The lab teaches source chain, urgency tactics, known-good routes, confidence before/after, and safe-use boundaries without relying on live scam links or partisan claims.

## Local command

```bash
python3 tools/verification-literacy-mini-lab/render_lab.py examples/education/verification_literacy_scam_lab_example.json
```

## Boundary

Use synthetic examples. Do not make learners click suspicious links, contact scammers, paste private messages into public tools, or treat this as cybersecurity, legal, financial, or identity-theft recovery training.
