# Shadow AI Amnesty Triage Sheet

A non-punitive intake sheet for one shadow-AI pattern. It asks what job the worker was trying to do, why approved tools failed, what data was touched, and what safer workaround or policy update is needed.

```bash
python3 tools/shadow-ai-amnesty-triage-sheet/render_sheet.py examples/team/shadow_ai_amnesty_triage_sheet_example.json
```
