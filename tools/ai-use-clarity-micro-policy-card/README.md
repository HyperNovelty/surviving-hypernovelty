# AI Use Clarity Micro-Policy Card

Translate a broad AI policy into one task-specific allowed/not-allowed/evidence/disclosure card.

## Use

```bash
python3 render_card.py ../../examples/education/ai_use_clarity_micro_policy_card_example.json
```

From the repo root:

```bash
python3 tools/ai-use-clarity-micro-policy-card/render_card.py examples/education/ai_use_clarity_micro_policy_card_example.json
```

## Boundary

Local-first review aid only. Do not use this as legal, compliance, security, medical, academic-integrity, or policy authority. Public posting, outreach, submissions, account changes, spend, real private data, and formal claims require separate human approval.
