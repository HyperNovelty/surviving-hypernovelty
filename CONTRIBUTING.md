# Contributing

This repository is being prepared as a safe public candidate. Contributions
should improve the local starter kit without adding private or proprietary
material.

## Before You Open a PR

Run:

```bash
python3 scripts/validate_repo.py
```

Use synthetic examples only. Keep paths repo-relative. Do not include private
Hypernovelty sources, transcripts, grant strategy, customer data, credentials,
student records, internal notes, or machine-specific home paths.

## Contribution Guidelines

- Prefer small, reviewable changes.
- Update examples or docs when tool behavior changes.
- Keep public claims modest: describe what the tool does locally, not adoption,
  funding, certification, or production readiness.
- Preserve human-review gates for public actions, accounts, money, medicine, law,
  schooling, customer data, or other high-stakes contexts.
- Use only Python standard-library dependencies unless maintainers explicitly
  approve otherwise.
