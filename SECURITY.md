# Security Policy

This repository is an early public candidate for a local starter kit. It contains
synthetic examples, schemas, templates, and small command-line tools for review.

## No Private Material

Do not add private Hypernovelty sources, transcripts, grant strategy, customer
data, credentials, account exports, private research notes, private medical or
education records, or internal filesystem paths.

Use repo-relative synthetic examples only.

## Local-First Boundary

The tools in this repository do not authorize:

- public publishing or outreach
- account login or account changes
- payment, funding, grant, legal, medical, or financial actions
- student/customer data handling
- production deployment or hosted services
- claims of adoption, endorsement, certification, or institutional approval

Outputs are review aids. Human approval is required before any public, account,
paid, high-stakes, or proprietary use.

## Reporting a Vulnerability

Report suspected security issues privately through the repository owner's normal
maintainer or security contact. Do not include secrets, private records, customer
data, or internal paths in public issues.
